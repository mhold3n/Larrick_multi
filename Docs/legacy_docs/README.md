# Legacy monolith notes (archived)

These Markdown files were copied from a retired local snapshot of the pre-split **Larrak** monolith (`Legacy_Monolithic/`, not tracked in this repo). They record historical paths and extraction ideas only. **Live code** for this repository lives under [`src/larrak2/`](../../src/larrak2/) and in the external **`larrak-core`**, **`larrak-simulation`**, and **`larrak-optimization`** packages (see root [`requirements-external.txt`](../../requirements-external.txt)). Do not treat paths here as current import paths.

This directory is named **`legacy_docs/`** (not `docs/legacy/`) so tracked files stay visible to Git on case-insensitive filesystems where `/Docs/` in `.gitignore` would otherwise hide a lowercase `docs/` path.
